# PSA Grabber Chrome Extension

**PSA Grabber** is a lightweight Chrome extension that allows users to access RSS quickly feeds for **HEVC-PSA movies** on [PSA.wf](https://psa.wf). With a single click on the "TORRENT" or "Download" links, the extension opens the corresponding RSS feed, saving you time and effort.

> **Note:** Currently supports **movies only**. TV show episode support is planned for a future update.

---

## Features

- Works seamlessly for **movies**.  
- Automatically detects the correct **movie title**.  
- Opens the **RSS feed** in a new tab for instant access.  
- Small on-screen **indicator** to show the grabber is active.  
- Non-intrusive: other clicks on the page function normally.
- Automatically copying Magnet links

4. Click **Load unpacked** and select the folder containing the extension files.  
5. Visit [PSA.wf](https://psa.wf) and check the **PSA Grabber indicator** at the bottom-right.  

---

## Usage

1. Navigate to a **movie page** on [PSA.wf](https://psa.wf).  
2. Click on the **TORRENT** or **Download** button.  
3. PSA Grabber will:  
   - Detect the title of the movie.  
   - Open the **Torrent RSS feed** for that release in a new tab.
   - Automatically Copying the magnet url

> TV show episode support will be added in a future update.
